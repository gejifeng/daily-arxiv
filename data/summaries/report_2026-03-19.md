## 1. Unified Spatio-Temporal Token Scoring for Efficient Video VLMs

**Authors**: Jianrui Zhang, Yue Yang, Rohun Tripathi et al.

**Categories**: cs.CV, cs.AI, cs.LG

**Link**: [2603.18004v1](https://arxiv.org/pdf/2603.18004v1)

**Summary**:
This paper introduces Spatio-Temporal Token Scoring (STTS), a unified method for pruning redundant vision tokens in video-based vision-language models (VLMs) to improve computational efficiency. Its key innovation is a lightweight module that scores and prunes tokens across both the vision transformer and the language model without needing text conditioning or complex merging, enabling end-to-end training. By learning spatial and temporal token importance via downstream gradients and an auxiliary loss, STTS achieves a 62% efficiency gain with only a 0.7% average performance drop on video QA tasks, with benefits scaling for longer videos.

## 2. Beyond Muon: MUD (MomentUm Decorrelation) for Faster Transformer Training

**Authors**: Ben S. Southworth, Stephen Thomas

**Categories**: cs.LG, math.NA, math.OC

**Link**: [2603.17970v1](https://arxiv.org/pdf/2603.17970v1)

**Summary**:
This paper introduces MUD (MomentUm Decorrelation), a new optimizer designed to accelerate transformer model training. It improves upon the Muon optimizer by replacing its computationally expensive polar decomposition step with a more efficient triangular whitening method inspired by Gram-Schmidt and Gauss-Seidel techniques. This key innovation reduces optimizer overhead substantially, leading to wall-clock speedups of 10-50% over tuned AdamW and Muon, with peak token throughput improvements of up to 3x on large models like GPT-2. The authors provide theoretical analysis, showing MUD converges to row-orthonormal matrices and exhibits quadratic local convergence, and validate its performance by matching Muon's perplexity on a protein language model in significantly less time.

## 3. Unified Policy Value Decomposition for Rapid Adaptation

**Authors**: Cristiano Capone, Luca Falorsi, Andrea Ciardiello et al.

**Categories**: cs.LG, q-bio.NC

**Link**: [2603.17947v1](https://arxiv.org/pdf/2603.17947v1)

**Summary**:
This paper introduces a unified policy-value decomposition framework for rapid adaptation in reinforcement learning. The core innovation is a bilinear actor-critic structure where both the policy and value functions share a low-dimensional, task-specific goal embedding. This allows the system to adapt to novel tasks instantly by recomputing this embedding via a single forward pass, without retraining the underlying neural network representations. The method, tested on a multi-directional locomotion task, enables specialized policy components to generalize to new goals through interpolation in the embedding space. The approach draws inspiration from biological gain modulation and offers a computationally efficient, biologically plausible mechanism for transfer learning in complex control systems.

## 4. Only relative ranks matter in weight-clustered large language models

**Authors**: Borja Aizpurua, Sukhbinder Singh, Román Orús

**Categories**: cs.LG, cs.CL

**Link**: [2603.17917v1](https://arxiv.org/pdf/2603.17917v1)

**Summary**:
This paper demonstrates that the relative ranking of weights—rather than their exact numerical values—is crucial for maintaining the performance of large language models (LLMs). By applying weight clustering to pretrained models like Llama 3.1-8B-Instruct, the authors show that reducing each weight matrix to just 16–64 distinct values preserves strong accuracy without retraining, offering a simple compression method. Experiments reveal that scrambling the relative ranks of clustered weights severely degrades model quality, while rank-preserving randomizations cause minimal loss. The study also identifies scale drift as a key failure mechanism when multiple layers are perturbed, though an affine correction can mitigate this. Overall, the work provides a rank-centric perspective for understanding model compression and robustness.

## 5. scicode-lint: Detecting Methodology Bugs in Scientific Python Code with LLM-Generated Patterns

**Authors**: Sergey V. Samsonau

**Categories**: cs.SE, cs.AI, cs.LG

**Link**: [2603.17893v1](https://arxiv.org/pdf/2603.17893v1)

**Summary**:
This paper introduces scicode-lint, a tool designed to detect methodology bugs—such as data leakage or incorrect cross-validation—in scientific Python code that produce plausible but erroneous results. Its key innovation is a two-tier architecture that separates pattern generation using frontier large language models (LLMs) at build time from efficient local model execution at runtime, eliminating the need for manual pattern engineering. This approach allows the tool to adapt to new library versions with minimal cost and demonstrates strong performance, achieving up to 97.7% accuracy on controlled tests and effective precision in real-world evaluations of Kaggle notebooks and published scientific papers.

## 6. RAMP: Reinforcement Adaptive Mixed Precision Quantization for Efficient On Device LLM Inference

**Authors**: Arpit Singh Gautam, Saurabh Jha

**Categories**: cs.LG, cs.AI

**Link**: [2603.17891v1](https://arxiv.org/pdf/2603.17891v1)

**Summary**:
This paper introduces RAMP, a reinforcement learning framework for adaptive mixed-precision quantization of large language models (LLMs) to enhance on-device inference efficiency. Its key innovation is an off-policy Soft Actor-Critic agent that assigns optimal per-layer bit-widths—conditioned on layer embeddings—to minimize perplexity under a global bit budget, enabling zero-shot transfer across model families. The method also incorporates Scale Folding, a preconditioning technique that stabilizes sub-4-bit quantization by migrating activation outliers into weights. Evaluated on Llama 2 7B, RAMP achieves better perplexity (5.54) at a smaller model size (3.68 GB) compared to uniform 4-bit methods, while policies trained on one model generalize effectively to others like Llama 2 13B and Mistral 7B. The resulting quantized models are exported via the HALO pipeline for efficient, kernel-free deployment across diverse hardware.

**Categories**: cs.LG, cs.AI

**Link**: [2603.17863v1](https://arxiv.org/pdf/2603.17863v1)

**Summary**:
This paper introduces DiscoGen, a procedural generator for creating diverse algorithm discovery tasks in machine learning, such as designing optimizers or loss functions. Its key innovation is addressing limitations of existing benchmarks—like data contamination and task similarity—by programmatically generating millions of tasks with configurable difficulty across various ML domains. The authors also present DiscoBench, a fixed subset of these tasks for standardized evaluation of algorithm discovery agents. ## 8. How do LLMs Compute Verbal Confidence

**Authors**: Dharshan Kumaran, Arthur Conmy, Federico Barbero et al.

**Categories**: cs.CL, cs.AI, cs.LG

**Link**: [2603.17839v1](https://arxiv.org/pdf/2603.17839v1)

**Summary**:
This paper investigates how large language models (LLMs) internally generate verbal confidence scores when prompted. Through experiments on models like Gemma 3 27B and Qwen 2.5 7B, the authors provide evidence that confidence is not computed just-in-time but is automatically cached during answer generation. Specifically, confidence representations emerge at positions adjacent to the answer and are stored for later retrieval when verbalized. The study further shows that these cached representations capture a richer evaluation of answer quality, explaining significant variance beyond simple token log-probabilities. This indicates that verbal confidence reflects a form of automatic self-evaluation rather than a post-hoc reconstruction, offering insights into LLM metacognition and calibration.

## 9. Text-to-Stage: Spatial Layouts from Long-form Narratives

**Authors**: Jefferson Hernandez, Swarnadeep Saha, Chenxi Whitehouse et al.

**Categories**: cs.CL, cs.AI, cs.LG

**Link**: [2603.17832v1](https://arxiv.org/pdf/2603.17832v1)

**Summary**:
This paper introduces Text-to-Stage, a framework for generating spatial stage-play layouts—including scenes, character positions, movements, and room types—from long-form narrative text that lacks explicit spatial descriptions. The key innovation is a training and inference method combining rejection-based supervised fine-tuning with reinforcement learning from verifiable rewards (GRPO), which enhances a language model's spatial reasoning capabilities. Evaluated on classical English literature, the approach outperforms baseline models in character attribution, spatial plausibility, and movement efficiency, aligning with both automated LLM judgments and human preferences. This work automates a complex dramaturgical process, offering potential benefits for downstream media and creative applications.

## 10. Symmetry-Reduced Physics-Informed Learning of Tensegrity Dynamics

**Authors**: Jing Qin, Muhao Chen

**Categories**: cs.LG

**Link**: [2603.17824v1](https://arxiv.org/pdf/2603.17824v1)

**Summary**:
This paper introduces SymPINN, a symmetry-reduced physics-informed neural network framework for modeling tensegrity dynamics. The key innovation is the explicit embedding of the structure's intrinsic geometric symmetries into the neural network architecture using group theory, which decomposes nodes into symmetry orbits and uses a reduced coordinate representation. This approach enforces equivariance through orbit-based coordinate generation and symmetry-consistent message passing, ensuring predicted configurations automatically satisfy symmetry constraints. The method also improves training by encoding initial conditions as hard constraints, using Fourier feature encoding, and employing a two-stage optimization strategy. Numerical experiments on T-bar and lander structures demonstrate that SymPINN achieves significantly higher prediction accuracy and computational efficiency compared to standard physics-informed models, highlighting the value of symmetry-aware learning for structure-preserving dynamic simulations.

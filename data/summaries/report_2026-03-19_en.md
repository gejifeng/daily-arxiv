## 1. Unified Spatio-Temporal Token Scoring for Efficient Video VLMs

**Authors**: Jianrui Zhang, Yue Yang, Rohun Tripathi et al.

**Categories**: cs.CV, cs.AI, cs.LG

**Link**: [2603.18004v1](https://arxiv.org/pdf/2603.18004v1)

**Summary**:
This paper introduces Spatio-Temporal Token Scoring (STTS), a novel method for improving the computational efficiency of video-based vision-language models (VLMs). Its key innovation is a unified, lightweight module that prunes redundant vision tokens across both the vision transformer and the large language model components without complex text conditioning or token merging. By learning to score tokens temporally via an auxiliary loss and spatially using downstream gradients, STTS achieves a 62% efficiency gain during training and inference by pruning 50% of vision tokens, with only a minimal 0.7% average performance drop across 13 video question-answering tasks. The method is fully end-to-end trainable and shows increasing benefits with more video frames, even enabling slight performance gains in long-video QA through test-time scaling.

## 2. Beyond Muon: MUD (MomentUm Decorrelation) for Faster Transformer Training

**Authors**: Ben S. Southworth, Stephen Thomas

**Categories**: cs.LG, math.NA, math.OC

**Link**: [2603.17970v1](https://arxiv.org/pdf/2603.17970v1)

**Summary**:
This paper introduces MUD (MomentUm Decorrelation), a new optimizer designed to accelerate transformer model training. It improves upon the Muon optimizer by replacing its computationally expensive polar decomposition step with a more efficient triangular whitening method inspired by Gram-Schmidt and Gauss-Seidel techniques. This key innovation reduces optimizer overhead substantially, leading to wall-clock speedups of 10-50% over AdamW and Muon, with peak token throughput improvements of up to 3x in some configurations. The authors provide a theoretical analysis showing MUD's convergence properties and demonstrate its practical effectiveness by matching Muon's validation perplexity in less time when training both GPT-2 and a 150M parameter protein language model.

## 3. Unified Policy Value Decomposition for Rapid Adaptation

**Authors**: Cristiano Capone, Luca Falorsi, Andrea Ciardiello et al.

**Categories**: cs.LG, q-bio.NC

**Link**: [2603.17947v1](https://arxiv.org/pdf/2603.17947v1)

**Summary**:
This paper introduces a unified policy-value decomposition framework for rapid adaptation in reinforcement learning. The core innovation is a bilinear actor-critic structure where both the policy and value functions share a low-dimensional, goal-conditioned coefficient vector, enabling zero-shot adaptation to novel tasks without retraining neural network representations. During pretraining, the method jointly learns a set of primitive policy and value basis functions, which are then frozen; at test time, adaptation to a new goal is achieved through a single forward pass to compute the new coefficients, reminiscent of biological gain modulation. Demonstrated on a multi-directional locomotion task for a MuJoCo Ant, the approach allows specialized policy components to generalize and interpolate for novel directions, offering a structured and efficient mechanism for transfer in complex control systems.

## 4. Only relative ranks matter in weight-clustered large language models

**Authors**: Borja Aizpurua, Sukhbinder Singh, Román Orús

**Categories**: cs.LG, cs.CL

**Link**: [2603.17917v1](https://arxiv.org/pdf/2603.17917v1)

**Summary**:
This paper demonstrates that the relative ordering of weights—rather than their exact numerical values—is crucial for maintaining the performance of large language models (LLMs). By applying weight clustering to pretrained models like Llama 3.1-8B-Instruct, the authors show that reducing each weight matrix to just 16–64 distinct values preserves strong accuracy without retraining, offering a simple compression method. Experiments reveal that scrambling the relative ranks of clustered weights severely degrades model quality, while rank-preserving perturbations cause minimal loss. The study also identifies scale drift as a key failure mode when multiple layers are altered, though an affine correction can mitigate this. These findings provide a new, rank-centric perspective on model compression and robustness.

## 5. scicode-lint: Detecting Methodology Bugs in Scientific Python Code with LLM-Generated Patterns

**Authors**: Sergey V. Samsonau

**Categories**: cs.SE, cs.AI, cs.LG

**Link**: [2603.17893v1](https://arxiv.org/pdf/2603.17893v1)

**Summary**:
This paper introduces scicode-lint, a tool designed to detect methodology bugs—such as data leakage or incorrect cross-validation—in scientific Python code, which often produce deceptively plausible but erroneous results. Its key innovation is a two-tier architecture that separates pattern generation using frontier large language models (LLMs) at build time from efficient pattern execution with a small local model at runtime, eliminating the need for manual engineering of detection rules. This approach allows the tool to adapt to new library versions automatically and scale with AI-generated code. Evaluations show promising performance, achieving up to 65% precision with full recall on Kaggle notebooks and 97.7% accuracy on controlled tests across 66 patterns, demonstrating its potential for automated verification in scientific software.

## 6. RAMP: Reinforcement Adaptive Mixed Precision Quantization for Efficient On Device LLM Inference

**Authors**: Arpit Singh Gautam, Saurabh Jha

**Categories**: cs.LG, cs.AI

**Link**: [2603.17891v1](https://arxiv.org/pdf/2603.17891v1)

**Summary**:
This paper introduces RAMP, a reinforcement learning framework for adaptive mixed-precision quantization of large language models (LLMs) to enhance on-device inference efficiency. Its key innovation is an off-policy Soft Actor-Critic agent that assigns optimal per-layer bit-widths—conditioned on activation, weight, and structural embeddings—to minimize perplexity under a global bit budget, enabling zero-shot transfer across model families. The method also incorporates Scale Folding, a preconditioning technique that stabilizes sub-4-bit quantization by migrating activation outliers into weights. Evaluations on Llama 2 7B show RAMP achieves better perplexity (5.54) at a smaller model size (3.68 GB) compared to uniform 4-bit methods, while policies trained on one model generalize effectively to others like Llama 2 13B and Mistral 7B, supporting the idea that quantization sensitivity is architecture-dependent.

## 7. Procedural Generation of Algorithm Discovery Tasks in Machine Learning

**Authors**: Alexander D. Goldie, Zilin Wang, Adrian Hayler et al.

**Categories**: cs.LG, cs.AI

**Link**: [2603.17863v1](https://arxiv.org/pdf/2603.17863v1)

**Summary**:
This paper introduces DiscoGen, a procedural generator for creating diverse algorithm discovery tasks in machine learning, such as designing optimizers or loss functions. Its key innovation is addressing limitations of existing benchmarks—like data contamination and task similarity—by procedurally generating millions of tasks with configurable difficulty across various ML domains. The authors also present DiscoBench, a fixed subset of these tasks for standardized evaluation of algorithm discovery agents. By enabling scalable and uncontaminated testing, DiscoGen opens new research directions for automating algorithm development and is demonstrated through prompt optimization experiments. The tool is released as open-source to support further advancements in the field.

## 8. How do LLMs Compute Verbal Confidence

**Authors**: Dharshan Kumaran, Arthur Conmy, Federico Barbero et al.

**Categories**: cs.CL, cs.AI, cs.LG

**Link**: [2603.17839v1](https://arxiv.org/pdf/2603.17839v1)

**Summary**:
This paper investigates how large language models (LLMs) internally generate verbal confidence scores when prompted to state their uncertainty. Through experiments on models like Gemma 3 27B and Qwen 2.5 7B, the authors provide evidence that confidence is not computed just-in-time but is automatically cached during answer generation and later retrieved for output. Key findings from activation steering and attention blocking show that confidence representations emerge at answer-adjacent positions and are gathered from answer tokens before being verbalized. Furthermore, linear probing reveals these cached representations capture richer information about answer quality beyond simple token log-probabilities. The study concludes that verbal confidence reflects an automatic, sophisticated self-evaluation process within LLMs, offering insights into their internal metacognitive mechanisms and calibration.

## 9. Text-to-Stage: Spatial Layouts from Long-form Narratives

**Authors**: Jefferson Hernandez, Swarnadeep Saha, Chenxi Whitehouse et al.

**Categories**: cs.CL, cs.AI, cs.LG

**Link**: [2603.17832v1](https://arxiv.org/pdf/2603.17832v1)

**Summary**:
This paper introduces Text-to-Stage, a framework for generating spatial stage-play layouts—including scenes, character positions, movements, and room types—from long-form narrative text that lacks explicit spatial descriptions. The key innovation is a training and inference method combining rejection-based supervised fine-tuning with reinforcement learning from verifiable rewards (GRPO), which enhances a language model's spatial reasoning capabilities. Evaluated on classical English literature, the approach outperforms baseline models in character attribution, spatial plausibility, and movement efficiency, aligning with both automated LLM judgments and human preferences. This work automates a complex dramaturgical task, demonstrating practical applications for media production and narrative analysis.

## 10. Symmetry-Reduced Physics-Informed Learning of Tensegrity Dynamics

**Authors**: Jing Qin, Muhao Chen

**Categories**: cs.LG

**Link**: [2603.17824v1](https://arxiv.org/pdf/2603.17824v1)

**Summary**:
This paper introduces SymPINN, a symmetry-reduced physics-informed neural network framework for modeling tensegrity dynamics. The key innovation is the explicit incorporation of the structure's intrinsic geometric symmetries into the neural network architecture using group theory, which decomposes nodes into symmetry orbits and uses a reduced coordinate representation. This approach enforces equivariance through orbit-based coordinate generation and symmetry-consistent message passing, ensuring predicted configurations automatically satisfy symmetry constraints. The method also improves training by encoding initial conditions as hard constraints, using Fourier feature encoding, and employing a two-stage optimization strategy. Experiments on symmetric structures demonstrate that SymPINN achieves significantly higher prediction accuracy and computational efficiency compared to standard physics-informed models, highlighting the value of symmetry-aware learning for structure-preserving dynamic modeling.
